# Design Guidelines - INEMA Academia

## Direcao visual
- Tema claro com foco em legibilidade para longos periodos de estudo.
- Tom visual: educativo, confiavel e energico.
- Tipografia principal: Inter.
- Tipografia de titulos e apoio didatico: Source Serif 4.

## Paleta base
- Primaria: #0284c7
- Secundaria: #2f855a
- Sucesso: #16a34a
- Erro: #dc2626
- Aviso: #eab308

## Cores por materia
- Matematica: #0ea5e9
- Ciencias: #16a34a
- Humanas: #7c3aed
- Linguagens: #ea580c
- Redacao: #db2777

## Componentes
- Cantos arredondados medios (14px).
- Cartoes com sombra suave e contraste alto.
- Indicadores de foco sempre visiveis para acessibilidade.

## Animacoes
- Feedback de acerto/erro deve ser curto (< 400ms).
- Confetti e Lottie so em momentos de conquista (nao em interacoes comuns).

## Mascote
- Recomendada criacao de mascote vetorial simples para onboarding e celebracoes.
- Entrega sugerida: pacote SVG em `public/mascot/`.
